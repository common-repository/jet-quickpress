== Description ==

EN: This plugin allows the users to write simple posts outside the dashboard (just like QuickPress from the Dashboard). With Tiny MCE Editor.

It adds a "Quickpress" submenu under /blog from which you can easily add a post to any of your blogs.

RU: Плагин дает возможность пользователям блогов размещать записи без использования адинистративной панели Wordpress. Создание записей с использованием редактора Tiny MCE.

(Based on BuddyPress Quick Press 0.1.6) 


== Installation ==

* Copy the directory jet-quickpress to wp-content/plugins.
* Activate the plugin site-wide
